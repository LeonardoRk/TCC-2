Em tecnologia usaremos o react native para criar a aplicação e depois será criado o apk para android.
As instruções utilizadas para instalar as ferramentas e gerar o build estão disponíveis na página do react native: https://facebook.github.io/react-native/docs/getting-started.html

Como pré requisito é preciso instalar:
NODE
react-native-cli  "npm install -g react-native-cli"
java development kit (jdk) versão 8 ou mais "sudo apt-get install openjdk-8-jdk"
android studio


Quando rodar o comando react-native run-android, caso de erro de não encontrar o SDK, é necessário mostrar ao sdk.dir onde está instalado o SDK do android. Para isso dentro da pasta android no projeto, crie um arquivo chamado local.properties, e nesse arquivo insira a linha sdk.dir =/path/to/Android/Sdk, que no meu caso é /home/leonardo/Android/Sdk
